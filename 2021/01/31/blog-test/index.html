安装 hexo

hexo官网 hexo

  1. npm install hexo-cli -g

  2. 安装完毕 hexo -v

  3. 安装 Hexo 完成后，再执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

  4. 指定文件目录结构

  5. ```hexo server
    cd myBlog
    hexo server

    $ git config –global user.name “用户名”
    $ git config –global user.email “邮箱地址”

    $ ssh-keygen -t rsa -C ‘上面的邮箱’

    $ cat ~/.ssh/id_rsa.pub

    ssh -T git@github.com

  6. 第二：要安装一个部署插件 hexo-deployer-git。

  7. 最后执行以下命令就可以部署上传啦，以下 g 是 generate 缩写，d 是 deploy 缩写：

    编写完markdown文件后，根目录下输入hexo g生成静态网页，然后输入hexo s可以本地预览效果，最后输入hexo d上传到github上

写文章

hexo new '文章名称' 在 /source/_post 下创建 .md文件

hexo g 生成静态文件

hexo s可以本地预览效果

hexo d上传到github上

hexo clean 清楚缓存文件 db.json和静态文件 public

hexo new draft "文章标题"，生成后会在 /source/_drafts 里看到你的草稿文章

hexo publish [layout] <filename> 将草稿文件发布到博客

hexo s --debug 开启调试模式
