    编写完markdown文件后，根目录下输入hexo g生成静态网页，然后输入hexo s可以本地预览效果，最后输入hexo d上传到github上

写文章

hexo new '文章名称' 在 /source/_post 下创建 .md文件

hexo g 生成静态文件

hexo s可以本地预览效果

hexo d上传到github上

hexo clean 清楚缓存文件 db.json和静态文件 public

hexo new draft "文章标题"，生成后会在 /source/_drafts 里看到你的草稿文章

hexo publish [layout] <filename> 将草稿文件发布到博客

hexo s --debug 开启调试模式
